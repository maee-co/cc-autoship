# データプライバシー・個人情報保護ポリシー

## 目的と適用範囲

本ドキュメントは、Claude Code 環境における個人情報（PII: Personally Identifiable Information）の取り扱いルールを定義します。プロジェクトの実情に合わせてカスタマイズしてください。

**適用対象**:
- リポジトリ内のすべてのコード・設定ファイル
- Claude Code がアクセスするコンテキストファイル
- 公開 API / 内部 API
- 環境変数・シークレット管理
- ログ・エラーハンドリング

---

## PII 定義と分類

### Level 1（厳重管理）

**絶対に公開 API・ログに含めてはいけない情報**

| カテゴリ | 具体例 |
|---------|--------|
| 識別情報 | 氏名（フルネーム）、生年月日、マイナンバー |
| 連絡先 | 電話番号、個人メールアドレス |
| 住所 | 市区町村より詳細（番地、建物名等） |
| 認証情報 | パスワード、API Key、トークン、Cookie |
| 金融情報 | クレジットカード番号、銀行口座 |

**取り扱い**:
- ❌ 公開 API 経由での提供禁止
- ❌ コミット禁止（`.gitignore` 必須）
- ❌ ログ出力禁止
- ✅ 環境変数（`.env`）で管理

### Level 2（慎重管理）

**公開版には含めず、完全版のみに記載できる情報**

| カテゴリ | 具体例 |
|---------|--------|
| 家族詳細 | 子どもの名前・誕生日、配偶者情報 |
| 財務情報 | 具体的な年収・時給 |

### Level 3（一般管理）

**公開可能な情報**

| カテゴリ | 具体例 |
|---------|--------|
| 居住地 | 市区町村レベル |
| 経歴 | 会社名・役職・在籍期間 |
| スキル | プログラミング言語、資格 |

## 基本ルール

### `.gitignore` 必須パターン

```gitignore
# 環境変数
.env
.env.local
.env.*.local

# 認証情報
*.key
*.pem
credentials.json

# ログファイル
*.log
logs/
```

### 環境変数管理

- 本番環境変数は `.env` に、テンプレートは `.env.example` に
- `.env.example` はコミット可能（実際の値なし）
- シークレット値は Issue / PR / docs に直接記載しない

## 違反時の対応

### PII 漏洩時の緊急対応

1. 漏洩箇所を即座に無効化・revoke
2. git 履歴から削除（BFG Repo-Cleaner 等）
3. 影響範囲を確認
4. 関係者へ報告
